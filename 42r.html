what is internet?

The Internet is a global system that connects computers and devices, letting people share information, talk, and access websites and apps.

what is browser?

A browser is a software application that lets you access and view websites on the internet.

Example : Google Chrome, Mozilla Firefox, Microsoft Edge, Safari, and Opera.

what is Url?

A URL (Uniform Resource Locator) is the web address you type in a browser to visit a specific website or page

Example : www.google.com.

what is HTML?

HTML (HyperText Markup Language) is the main code used to create and structure content on web pages, like text, images, and links.

What is difference between Markup Language and Programming Language ?

Purpose: Markup Language is used to structure and format data (like text and images) on a web page. Programming Language is used to write instructions that a computer can execute to perform tasks.

Examples: Markup Language: HTML, XML Programming Language: Python, Java, C++

what is IP address?

An IP address (Internet Protocol address) is a unique number given to each device connected to the internet so it can be identified and communicate with other devices.

Example : IP address is 192.168.1.1 (used in many home networks) or 8.8.8.8 (a public IP address from Google).
